# ADR-0001: RTSP Recorder dengan Audio

## Status
Accepted

## Versi
v0.2.0

## Konteks
Sistem AI Director membutuhkan kemampuan mengambil rekaman dari kamera existing melalui RTSP tanpa mengganti hardware.

## Keputusan
Recorder menggunakan container ffmpeg melalui Docker Compose.
Video dan audio disimpan ke file MP4 dengan codec copy agar tidak membebani CPU.

## Konfigurasi
Semua secret RTSP disimpan di file .env.
Contoh konfigurasi tersedia di .env.example.

## Konsekuensi
- Bisa dijalankan ulang dengan docker compose up.
- Tidak ada install manual selain Docker/Git.
- Audio ikut terekam jika stream RTSP kamera menyediakan audio.
